# Web Navigator

智能、高效的网页导航工具，帮助你快速访问常用网站。

## 功能特点

- 🔗 **链接管理**：添加、编辑、删除常用网站链接
- 📂 **分类整理**：自定义分类，按需整理导航
- 🔍 **快速搜索**：实时过滤，快速找到目标网站
- 🌓 **主题切换**：支持深色/浅色模式
- 💾 **数据持久化**：本地存储，无需登录
- 📤 **导入导出**：支持数据备份和恢复

## 技术栈

- 纯 HTML + CSS + JavaScript
- Tailwind CSS CDN
- LocalStorage API
- HTML5 Drag & Drop

## 使用方法

1. 点击 "添加链接" 创建新的导航项
2. 拖拽卡片调整顺序
3. 使用搜索框快速查找
4. 点击设置图标管理数据

## 适用场景

- 个人浏览器主页
- 团队内部导航
- 常用工具快速访问
- 工作学习书签管理
