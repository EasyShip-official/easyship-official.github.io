# Web Navigator 🧭

一个智能、高效的网页导航工具，采用文件夹式设计，帮助你快速访问常用网站。

## 截图预览

- **文件夹卡片**：所有分类一目了然，点击展开查看链接
- **原地展开**：点击文件夹直接在网格中展开，无需跳转
- **炫酷视觉**：动态渐变背景、毛玻璃效果、流畅动画

## 功能特点

### 📁 文件夹式导航
- **一屏显示**：所有内容在视口内，无需滚动
- **点击展开**：点击文件夹直接展开所有链接
- **原地操作**：在展开区域内直接管理链接
- **状态记忆**：记住上次打开的文件夹

### 🔗 链接管理
- **快速添加**：支持自定义网站名称、URL、Emoji 图标
- **编辑删除**：悬停显示操作按钮，快速管理
- **直接访问**：点击链接直接打开网站（新标签页）

### 📂 分类管理
- **自定义分类**：创建个性化分类（工作、开发、工具等）
- **视觉标识**：每个分类独立的图标和主题色
- **灵活编辑**：随时修改分类名称、图标和颜色

### 🎨 视觉体验
- **动态背景**：渐变色动画 + 漂浮光球效果
- **毛玻璃卡片**：半透明背景 + 模糊效果
- **流畅动画**：悬浮、展开、缩放等过渡效果
- **深色模式**：支持明暗主题切换

### 💾 数据安全
- **本地存储**：所有数据保存在浏览器 LocalStorage
- **导入导出**：支持 JSON 格式备份和恢复
- **隐私保护**：数据不上传服务器，完全本地化

## 技术栈

- **HTML5** - 语义化标签
- **Tailwind CSS CDN** - 样式框架
- **Vanilla JavaScript** - 无依赖原生 JS
- **LocalStorage API** - 数据持久化

## 使用方法

### 基本操作

1. **访问链接**
   - 点击文件夹卡片展开
   - 点击链接直接访问（新标签页打开）

2. **添加链接**
   - 展开目标文件夹
   - 点击右上角 ➕ 按钮
   - 填写网站信息并保存

3. **管理链接**
   - 展开文件夹
   - 悬停在链接上
   - 点击编辑或删除按钮

4. **管理分类**
   - 展开目标文件夹
   - 点击右上角 ✏️ 按钮编辑分类
   - 或点击顶部「新建分类」创建新分类

5. **切换主题**
   - 点击顶部 🌙/☀️ 按钮切换深色/浅色模式

### 快捷键

- `ESC` - 收起已展开的文件夹 / 关闭模态框

## 常见问题

**Q: 数据会丢失吗？**
A: 数据保存在浏览器 LocalStorage 中，除非清除浏览器数据，否则不会丢失。建议定期使用导出功能备份。

**Q: 可以在不同设备间同步吗？**
A: 目前不支持自动同步，但可以通过导出功能手动备份数据，然后在其他设备导入。

**Q: 如何恢复默认数据？**
A: 点击设置按钮 → 重置所有数据，可以恢复到初始状态。

**Q: 支持多少个链接？**
A: 理论上无限制，但建议每个分类不超过 20 个链接以保持良好的使用体验。
