# 网页导航项目 - 技术规格说明

## 项目概述

一个智能、高效的网页导航工具，帮助用户快速访问常用网站。

## 核心功能

### 1. 导航链接管理
- **添加链接**：支持添加网站名称、URL、图标、分类
- **编辑链接**：修改已有链接的信息
- **删除链接**：移除不需要的链接
- **拖拽排序**：通过拖拽调整链接顺序

### 2. 分类管理
- **创建分类**：自定义链接分类（工作、学习、娱乐等）
- **分类图标**：为不同分类设置图标
- **分类颜色**：支持自定义分类颜色

### 3. 搜索功能
- **实时搜索**：输入关键词快速过滤链接
- **模糊匹配**：支持网站名称和 URL 模糊搜索

### 4. 数据持久化
- **LocalStorage**：使用浏览器本地存储保存数据
- **导入/导出**：支持数据备份和恢复

### 5. 视觉设计
- **响应式布局**：适配桌面和移动设备
- **深色/浅色模式**：支持主题切换
- **快捷方式**：点击图标直接跳转

## 技术栈

- **HTML5**：语义化标签
- **Tailwind CSS CDN**：样式框架
- **Vanilla JavaScript**：无依赖原生 JS
- **LocalStorage API**：数据持久化
- **HTML5 Drag & Drop API**：拖拽排序

## 交互流程

1. **添加链接**
   - 点击 "添加链接" 按钮
   - 填写链接信息
   - 选择分类
   - 保存

2. **拖拽排序**
   - 长按链接卡片
   - 拖动到目标位置
   - 释放并保存新顺序

3. **搜索**
   - 在搜索框输入关键词
   - 实时过滤显示匹配结果
   - 清空搜索恢复完整列表
