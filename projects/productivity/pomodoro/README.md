# Pomodoro Timer

简洁实用的番茄工作法计时器，帮助你保持专注。

## 功能特点

- 标准番茄工作法计时（25分钟工作，5分钟休息）
- 长休息提醒（每4个番茄后15分钟休息）
- 可自定义工作/休息时长
- 浏览器通知提醒
- 计时历史记录
- 统计数据可视化

## 技术栈

- Vue.js 3
- Tailwind CSS
- LocalStorage 数据持久化
- Web Notifications API

## 使用方法

1. 点击"开始"按钮启动25分钟工作计时
2. 计时结束后，开始5分钟短休息
3. 完成4个番茄后，享受15分钟长休息
4. 查看统计数据了解你的专注时间

## 番茄工作法

番茄工作法是一种时间管理方法：
- 工作25分钟，休息5分钟
- 每4个番茄后，休息15-30分钟
- 有助于提高专注力和工作效率
