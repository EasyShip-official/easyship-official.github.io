# The Lucidity Clock

数字健康仪表盘，用于情感 detachment 与薪资可视化。

## 功能特点

- 赛博朋克风格的视觉设计
- 实时薪资计算显示
- 工作时间与生活平衡可视化
- 情感 detachment 提醒
- 数字健康追踪

## 技术栈

- Vue.js 3
- Tailwind CSS
- LocalStorage 数据持久化
- 动态数据可视化

## 使用方法

1. 设置你的时薪或月薪
2. 设置工作时间和非工作时间
3. 观看实时薪资变化
4. 反思工作与生活的平衡

## 设计理念

这是一个实验性的数字艺术项目，旨在：
- 探讨工作的价值与时间的意义
- 通过可视化促进自我反思
- 提供独特的赛博朋克美学体验
