# AI Chat Assistant

一个基于 OpenAI API 的极简聊天界面。

## 功能特点

- 连接 OpenAI API 进行智能对话
- 简洁直观的用户界面
- 支持多轮对话上下文记忆
- 本地存储聊天历史记录

## 技术栈

- 纯 HTML + CSS + JavaScript
- OpenAI API
- LocalStorage 数据持久化

## 使用方法

1. 获取 OpenAI API Key
2. 在应用界面中输入 API Key
3. 开始对话

## 注意事项

- API Key 仅存储在本地浏览器中
- 请勿在公共设备上保存 API Key
